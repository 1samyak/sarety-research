# SARETY Research

Professional equity research website with institutional-grade design inspired by Morgan Stanley, Morningstar, and Bernstein.

## Features

- Premium Financial Times–style paper theme
- Company snapshot sidebar with key metrics
- BUY/HOLD/SELL recommendation badges
- Key metrics strip (CMP, Target, Upside, Market Cap, P/E)
- Interactive Table of Contents
- Drop-cap article styling
- Running headers and page numbers
- Print-to-PDF support (A4 optimized)
- Dark/Light mode toggle
- Responsive design
- Research disclaimers

## Project Structure

```
sarety-research/
├── index.html          # Landing page
├── report.html         # Full equity research report
├── css/
│   ├── style.css       # Global styles
│   ├── report.css      # Report-specific styles
│   └── print.css       # Print/PDF styles
├── js/
│   ├── app.js          # Main app functionality
│   └── report.js       # Report functionality
└── README.md
```

## Getting Started

Simply open `index.html` in a web browser to view the website.

## Usage

- Navigate between pages using the menu
- Toggle dark/light mode with the theme button
- Download reports as PDF using the download button
- Print-friendly styling for high-quality output
